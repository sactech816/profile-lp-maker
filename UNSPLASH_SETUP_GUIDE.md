# Unsplash API 画像自動選定機能 セットアップガイド

## 概要
エディタの「自動」ボタンをクリックすると、タイトルに基づいてUnsplashから関連する画像を自動検索・選定できる機能です。

## セットアップ手順

### 1. Unsplash APIキーの取得

1. [Unsplash Developers](https://unsplash.com/developers)にアクセス
2. 「Register as a developer」をクリック
3. アカウントを作成（既にある場合はログイン）
4. 「New Application」をクリック
5. 利用規約に同意してアプリケーションを作成
6. アプリケーション名と説明を入力
7. 「Access Key」をコピー

### 2. 環境変数の設定

`.env.local`ファイルに以下を追加:

```env
UNSPLASH_ACCESS_KEY=your_access_key_here
```

### 3. 動作確認

1. サーバーを再起動
2. エディタの「基本設定」ステップに移動
3. タイトルを入力（例: "ビジネス 成功"）
4. 「自動」ボタンをクリック
5. 関連する画像が表示されるモーダルが開く
6. 好きな画像を選択

## 注意事項

### APIキーなしでも動作します
- 環境変数が設定されていない場合、キュレーション済みの画像がランダムに表示されます
- 機能は正常に動作しますが、タイトルに基づいた検索はできません

### Unsplash利用規約
- Unsplashの画像を使用する場合、クレジット表記が推奨されます
- 商用利用可能ですが、利用規約を確認してください
- [Unsplash License](https://unsplash.com/license)

### API制限
- 無料プランでは1時間あたり50リクエストまで
- 本番環境では有料プランの検討を推奨

## トラブルシューティング

### 「画像の検索に失敗しました」と表示される
- 環境変数が正しく設定されているか確認
- サーバーを再起動
- APIキーが有効か確認

### 画像が表示されない
- ブラウザのコンソールでエラーを確認
- ネットワーク接続を確認
- Unsplash APIの制限に達していないか確認

## 機能の特徴

✅ タイトルに基づいた自動画像検索
✅ 6枚の候補から選択可能
✅ プレビュー表示
✅ Unsplashクレジット表示
✅ APIキーなしでも動作（デフォルト画像）

## 実装ファイル

- `app/api/search-images/route.js` - 画像検索API
- `components/Editor.jsx` - エディタコンポーネント（画像選択モーダル）
